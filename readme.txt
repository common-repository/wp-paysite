WP-Paysite allows you to run a adult membership based site on WordPress platform.

== Description ==

This free PaySite script gives you a possibility you to run a full-featured 
adult membership based site on WordPress platform. It covers both front-end 
and back-end functionality preserving generic WordPress features.

We are glad to present you the first ever free PaySite script. With WP-PaySite 
you can save several grands on buying the PaySite script or paying any monthly 
fees.

= What does it cover =

* Handles both photo and video updates for the front-end.
* Automatically crops thumbnails for both front-end and members area
* Main update image for photo updates and adjustable number of additional images
* Integrated flash player for video updates display at the front-end.
* Any additional info pages for the front-end.
* Manages photo and video updates at the member area
* Possibility to categorize updates
* Dynamic or static front-end
* Multilingual support
* Multiple models support with personal pages for each model
* Configurable number of tour pages and updates per page
* Possibility of direct integration into members area

= What is not covered =

The initial idea was to make a solution that have all adult paysite features. 
However most of you are rather skeptic about WordPress. So user authentication 
is not covered. Nevertheless WordPress Paysite Plug-in doesn't conflict with 
any member authentication system.
